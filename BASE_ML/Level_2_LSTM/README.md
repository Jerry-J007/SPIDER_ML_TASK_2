## Base Task 2: Custom LSTM for Weather Forecasting
This module involves building and evaluating a custom Long Short-Term Memory (LSTM) model tailored for time-series climate prediction. 

**Key Objectives:**
*   **Architecture Design:** Construct a custom LSTM network optimized for sequential weather data.
*   **Model Training:** Train the network to forecast future weather patterns based on historical metrics.
*   **Performance Analysis:** Conduct an in-depth evaluation of the model's accuracy, architectural efficiency, and forecasting capabilities.
